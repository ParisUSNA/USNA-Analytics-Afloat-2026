# APP Basics

## How to compile the app
> **Do not push the compiled app to git!!!**
1. Option 1: Standard/Quiet
    - make
2. Option 2: Loud
    - make loud


## How to run the app
1. cd /dist
2. ./run_app

## Notes on editing the app
- The target file for compilation is app.py
- The app uses streamlit and pandas for its displays
- run_app.py is moody, please only edit to improve or fix
